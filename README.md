# Café Aroma - React

Este repositorio contiene el código fuente de la aplicación web interactiva Café Aroma, desarrollada con React y Vite para la gestión y simulación de pedidos en una cafetería. Proyecto académico desarrollado para la Universidad Tecnológica del Perú (UTP).

---

## Enlace del Proyecto Desplegado
La aplicación se encuentra disponible en producción a través del siguiente enlace:
* **URL de Vercel:** [https://cafe-aroma-react.vercel.app](https://cafe-aroma-react.vercel.app)

---

## Características del Proyecto
* **Catálogo Dinámico:** Carga asíncrona de productos (cafés y postres) estructurados desde un archivo de configuración de datos local (db.json).
* **Gestión de Pedidos:** Simulación de carrito de compras con cálculo automatizado de subtotales, impuestos y montos totales en tiempo real.
* **Manejo de Estado Reactivo:** Control de flujo y disponibilidad de inventario utilizando Hooks de React (useState y useEffect).
* **Diseño Responsivo:** Interfaz adaptada para una correcta visualización en dispositivos móviles y de escritorio.

---

## Tecnologías Utilizadas
* **React (v18):** Librería principal orientada a componentes para la interfaz de usuario.
* **Vite:** Herramienta de construcción y entorno de desarrollo rápido.
* **JavaScript (ES6+):** Lenguaje de programación para la lógica de negocio.
* **HTML5 / CSS3:** Estructura y estilos personalizados de la aplicación.
* **Git / GitHub:** Control de versiones y alojamiento de código.
* **Vercel:** Hosting y despliegue continuo en la nube.
